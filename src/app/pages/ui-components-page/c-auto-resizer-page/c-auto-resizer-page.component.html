Auto Resizer

AutoResizerDirective is added to the <textarea> element to set the height variably depending on the value of the element. 

<app-form-field withBorder>
  <label>Auto Resizer</label>

  <textarea appAutoResizer></textarea>
</app-form-field>

Attributes

disableLineBreaks: boolean

You can ignore line breaks by setting the disableLineBreaks attribute. 

<app-form-field withBorder>
  <label>Auto Resizer</label>

  <textarea disableLineBreaks appAutoResizer></textarea>
</app-form-field>
