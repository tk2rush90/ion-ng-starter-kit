Bottom Actions

BottomActionsComponent is used to place action buttons at the bottom of modals, bottom sheets, and various other elements. 

<div class="p-4">Bottom Actions</div>

<app-bottom-actions justify="right">
  <button
    type="button"
    theme="red"
    appFlatButton>Cancel</button>

  <button
    type="button"
    appFlatButton>Confirm</button>
</app-bottom-actions>
Bottom Actions

Attributes

justify: 'right' | 'left' | 'between'

The justify attribute is a required. Arrange elements within a component to the 'left' or 'right' , or create gaps 'between' elements. 

<div class="p-4">Justify Left</div>

<app-bottom-actions justify="left">
  <button
    type="button"
    theme="red"
    appFlatButton>Cancel
  </button>

  <button
    type="button"
    appFlatButton>Confirm
  </button>
</app-bottom-actions>

withBorder: boolean

The withBorder attribute sets the border at the top of BottomActionsComponent .
